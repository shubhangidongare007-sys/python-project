# Python Mini Projects

## Description
This repository contains Python mini projects developed for academic purposes. These projects demonstrate the use of Python programming concepts such as functions, loops, dictionaries, file handling, and exception handling.

## Projects Included
- Expense Tracker
- Inventory Management System
- Library Management System
- Quiz System
- Student Management System

## Technologies Used
- Python 3

## How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the required `.py` file.
